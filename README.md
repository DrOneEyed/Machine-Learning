# Machine Learning — Algorithm Implementations

A practical notebook collection covering core machine learning algorithms and real-world NLP datasets.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `main.ipynb` | Core ML algorithm implementations — regression, classification, clustering |
| `DataHunt.ipynb` | Data exploration, cleaning, and feature engineering pipeline |
| `ML_Corona_NLP_Dataset.ipynb` | Sentiment analysis and text classification on a COVID-19 NLP dataset |

## Getting Started

```bash
git clone https://github.com/DrOneEyed/Machine-Learning.git
cd Machine-Learning
pip install scikit-learn pandas numpy matplotlib seaborn nltk jupyter
jupyter notebook
```

## Algorithms Covered

- Linear & Logistic Regression
- Decision Trees & Random Forests
- K-Nearest Neighbours
- Support Vector Machines
- K-Means Clustering
- Natural Language Processing (NLP) with TF-IDF & sentiment classification
